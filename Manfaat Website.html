Manfaat Website

Di awal pembuatannya website hanya dimanfaatkan untuk alat pertukaran informasi para ilmuwan di perguruan tinggi. Pada tahun 1994, website dimanfaatkan untuk tujuan lain seperti bisnis (Microsoft, IBM, dan Pizza Hut), organisasi (Amnesty International), lembaga pemerintah (White House), situs berita (The Telegraph dan The Economist), dan mesin pencari (Yahoo!).

Kini website tidak eksklusif untuk perusahaan besar, organisasi ternama, atau pemerintah adidaya saja. Bahkan website dapat dimanfaatkan untuk tujuan pribadi seperti berbagi pengalaman perjalanan, menulis tutorial, atau menampilkan karya dan portofolio.
