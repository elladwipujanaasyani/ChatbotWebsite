# Chatbot di File Tugas Akhir
Proyek ini bertujuan untuk membuat chatbot berbasis web yang memberikan informasi tentang website.

## Struktur Folder
  - `index.html`: Halaman utama proyek.
  - `assets/`: Folder berisi file pendukung.
  - `css/`: File CSS untuk desain.
  - `images/`: Gambar untuk chatbot.
  - `js/`: Logika JavaScript.
  - `data/`: Database chatbot dalam format CSV.

## Cara Menjalankan
1. Buka file `index.html` di browser.
2. Pastikan semua file pendukung ada di lokasi yang benar.
